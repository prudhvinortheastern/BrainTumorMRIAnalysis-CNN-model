# BrainTumorMRIAnalysis-CNN-model
Trained 4 CNN architectures (Baseline CNN, Deeper CNN, ResNet-18,  EfficientNet-B0) on 7,200 brain MRI images for 4-class tumor  classification using PyTorch. Applied transfer learning, data  augmentation, and BatchNorm regularization; evaluated via F1-Score,  ROC-AUC, and confusion matrices.
